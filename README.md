# CHALLENGER-01 ALURA STORE
  Challenge de Alura para desarrollar habilidades de Análisis de Datos y Ciencia de Datos

# Objetivo:
  Apoyar al Sr. Juan decidir y salir de la incognita, "Cuál tienda debe vender?", para que pueda invertir. 
  Se realizo un analisis completo con los datos historicos registrados de las cuatro tiendas que posee, en base a esos datos se realizo     el analisis y se obtuvubo lo siguiente.

# Resultados:

  💰 Los ingresos totales de las tiendas.

  🚚 Las categorías de productos más y menos vendidas.

  ⭐ Las calificaciones promedio de los clientes por tienda.

  🗺️ Los productos más y menos vendidos.

  📉 El coste de envío promedio para cada tienda.
